User Help
The AR-semble robotic assembly assistance system includes a user help mechanism designed to support smooth and intuitive interaction with the application, even for users with no prior technical experience. User assistance is primarily provided through an intuitive interface design, clear on-screen guided steps, and flexible interactive 3D controls.

Download the AR-semble mobile application and allow installation from unknown sources. Upon launching the application, grant permissions for the camera. Users are guided naturally through a step-by-step assembly manual interface, where each mechanical progression page displays the exact real-time structural layout of the LEGO NXT robot parts. Users can easily play, pause, or rewind animations, inspect components from multiple angles, and click backward to review structural joints.

Once the physical robot build is completed successfully, the system unlocks an interactive Drive Mode game reward. This allows students to control their physical robot through a virtual interface, that turn a technical engineering task into a highly engaging, gamified milestone.

To open the project on your Unity Editor, follow these steps: 
1. Download the project repository from GitHub and extract the source folder on your computer.
2. Open Unity Hub, click Add project from disk, select the extracted source file folder root, and click open.
3. It may take some time in order for the Unity Engine to resolve packages, import 3D models, and fully build the project caching on your computer.
4. Once the project opens, go to File -> Build Settings, select Android from the platform listing, and click Switch Platform.
5. Locate the main assembly scene under Assets/Scenes/, double-click to load it, and press the Play button in the upper toolbar to test and simulate the application.
